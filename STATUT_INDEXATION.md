1. **Dans Google Search Console** :
   - Section "Couverture" > "Valide"
   - Vous verrez les pages indexées

2. **Recherche Google** :
   - Recherchez : `site:pixeltech-antilles.com`
   - Vous verrez toutes les pages indexées

3. **Recherche par nom** :

   - Le site devrait apparaître une fois indexé
