# Soumission du sitemap dans Google Search Console

## URL du sitemap

Votre sitemap est accessible à : `https://pixeltech-antilles.com/sitemap.xml`

## Étapes dans Google Search Console

1. Dans la section **"Sitemaps"** (menu de gauche)
2. Dans le champ **"Ajouter un nouveau sitemap"**
3. Entrez simplement : `sitemap.xml`
   - Google ajoute automatiquement le domaine
   - Pas besoin de l'URL complète
4. Cliquez sur **"Envoyer"**

## Après soumission

- Google va crawler le sitemap automatiquement
- Les pages seront indexées progressivement
- Vous pouvez suivre l'avancement dans Search Console

## Pages dans le sitemap

Votre sitemap contient actuellement :
- `/` - Page de construction (noindex)
- `/home` - Page d'accueil principale
- `/services` - Services
- `/tarifs` - Tarifs
- `/a-propos` - À propos
- `/contact` - Contact

## Vérification

Vous pouvez vérifier que le sitemap est accessible en allant sur :
`https://pixeltech-antilles.com/sitemap.xml`

Vous devriez voir le XML avec toutes les URLs listées.
