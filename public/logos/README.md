# Logos PixelTech

Placez vos logos dans ce dossier.

## Structure recommandée

- `logo.svg` - Logo principal (horizontal)
- `logo-icone.svg` - Icône carrée pour favicon
- `logo-horizontal.svg` - Logo horizontal compact

## Utilisation

Une fois vos logos placés ici, décommentez les lignes dans `src/components/Header.tsx` pour les utiliser.

Pour le favicon, placez `logo-icone.svg` ou `logo-icone.png` à la racine de `public/` et référencez-le dans `src/app/layout.tsx`.
